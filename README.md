# 文字处理工具

> 配合 [PoxenStudio/mybooks](https://github.com/PoxenStudio/mybooks) 项目使用，用于快速处理图书简介文本，一键生成 HTML 格式。

## 功能

- **文本转 HTML** — 按空行自动分段，每段包裹 `<p>` 标签，段内换行转 `<br>`
- **自动实时处理** — 输入文字即时处理，无需手动点击按钮
- **关键词高亮** — 自定义关键词自动加粗、加大（`<h2>`）、高亮颜色
- **正则表达式规则** — 支持正则匹配文本并应用样式（如 `\d+` 匹配数字、`第.+章` 匹配章节）
- **书名号加粗** — `《书名》` 自动用 `<b>` 标签加粗
- **实时预览** — 左侧输入，右侧即时预览处理效果
- **HTML 代码输出** — 底部面板显示并可直接复制处理后的 HTML 代码
- **首行缩进** — 设置面板可调节段落首行缩进（0~4em）
- **面板拖拽** — 左右面板宽度可拖拽调整，最小宽度 200px
- **规则导入/导出** — 支持导出规则为 JSON 文件，方便备份与团队共享
- **配色切换** — 深色 / 浅色 / 跟随系统三种主题
- **纯离线可用** — 单个 HTML 文件，无需网络，双击即用

## 使用方式

1. 在左侧文本框粘贴图书简介文字（每段用空行分隔）
2. 右侧即时显示处理后的预览效果
3. 底部查看 HTML 代码，点「复制HTML」即可复制
4. 复制后的代码可直接用于 mybooks 项目

## 设置面板

点击 ⚙️ 按钮展开设置面板，可调节：

- **预览字号** — 可调范围 12~24px
- **段落间距** — 可调范围 0~3em
- **首行缩进** — 可调范围 0~4em
- **高亮颜色** — 自定义颜色选择器
- **关键词规则** — 自定义添加/删除关键词匹配规则
- **正则规则** — 自定义添加/删除正则表达式匹配规则

## 默认关键词规则

| 关键词 | 样式 |
|--------|------|
| 重要 | 加粗 |
| 注意 | 加粗+加大（`<h2>`）|
| 警告 | 高亮颜色 |
| 提示 | 加粗 |
| 关键 | 加粗+加大（`<h2>`）|

可在设置面板中自定义添加/删除规则。

## 处理顺序

1. HTML 转义 → 2. 正则规则 → 3. 关键词规则 → 4. 书名号加粗

正则规则先于关键词规则执行，规则按关键词长度降序匹配。

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl + S` | 复制 HTML |
